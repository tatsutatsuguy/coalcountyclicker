About The Game:

Coal County Clicker is a humorous game based on the popular idle-clicker game genre (cookie clicker, hero clicker). The intent is to have fun with JavaScript, and show off the complexity of aggregatting math functions, and my ability to manipulate them as a developer, in an engaging way.

This game is set on EASY mode, for the intent of showing off what it can do. Typically the payouts would take much longer in a clicker game, but I'm assuming whomever would look over this would want to see what it can do in a few minutes, not hours or days. If you want to see my work please visit FurnaceStreetStudio.
